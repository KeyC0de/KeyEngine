In DirectX 11.1+ you can get helpful information from DX Device Context using the DXGIGetDebugInterface given we have the device debug layer enabled

The confusing part is that the function you need, DXGIGetDebugInterface is not defined in any header and is not present in any import library.

So we have to load it dynamically
