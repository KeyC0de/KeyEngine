\doxysubsection{Detailed Description}
\doxylink{classcon_1_1CBElement}{CBElement} instances form a tree that describes the layout of the data buffer. 

=============================================================

\begin{DoxyAuthor}{Author}
Key\+C0de 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2022/08/21 19\+:55
\end{DoxyDate}
supporting nested aggregates of structs and arrays

friend relationships are used liberally throught the Dynamic\+Constant system

instead of seeing the various classes in this system as encapsulated decoupled units,

they must be viewed as aspect of one large monolithic system

the reason for the friend relationships is generally so that intermediate

classes that the client should not create can have their constructors made private, \doxysubsection*{so that cook\+Layout() cannot be called on arbitrary CBElements, etc. }
